[HTML]
기본 구조

((위 화면은 VS code 에서 '!' (느낌표) 입력 후 tab 키를 눌렀을 때 나타나는 기본 구조 입니다.))

1. <!DOCTYPE html>
 : 웹페이지에 사용된 HTML 의 종류와 버전 지정
문서 유형 선언 부분, 현재 문서는 HTML5 언어로 작성된 웹 문서

2. <html>
: 웹 문서의 시작과 끝을 나타냄

3. <head>
1) <meta charset="UTF-8">
: 브라우저에서 표시되는 과정에서 인코딩 할 때 지정된 문자코드로 인코딩 하라는 뜻

  • 특별한 이유가 없는 한 HTML5 문서에서는 대부분 UTF-8 로 지정
  • 한국어, 영어, 일본어만 사용할 경우에는 euc-kr 을 선언하기도 하지만 권장하지 X
2) <meta name="viewport" content="width=device-width, initial-scale=1.0">
: 사용자의 보이는 영역이 기기별로 다르므로 이것을 지정해주는 태그
우리가 개발을 할 때는 딱히 신경쓰지 않아도 됨

3) <title>
: 탭 제목

4. <body>
: 실제 브라우저 화면에 표시될 내용
