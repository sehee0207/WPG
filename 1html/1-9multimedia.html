HTML5 부터 <audio>, <video> 태그 사용 가능

1. <audio>
1) src : 재생할 오디오가 존재하는 URL 지정
2) autoplay : 음악 자동 재생
3) controls : 브라우저가 오디오 재생 제어기 표시
4) loop : 반복 재생
5) muted : 음소거
6) <source> : 하나의 오디오 소스에 여러 형식의 미디어 파일을 한번에 지정해주는 태그, 호환성 증가
* 재생 가능한 오디오 파일 형식에는 제한이 있음

2. <video>
1) 속성은 audio 와 거의 동일하다
2) src, autoplay, controls, loop, muted
3) poster : 재생 전이나 불가 시 표시되는 이미지
4) width, height : 비디오 너비, 높이
5) <source>
* 재생 가능한 비디오 파일 형식에는 제한이 있음

3. 인라인 프레임
: 현재 html 페이지 내에 내장 윈도우를 만들어 다른 html 페이지를 출력 할 수 있도록 해줌
<iframe> 태그에 의해 만들어지는 윈도우 -> 인라인 프레임
1) src : 프레임에 표시할 문서의 URL 지정
2) width, height : 너비, 높이
3) name : 인라인 프레임 윈도우의 이름
