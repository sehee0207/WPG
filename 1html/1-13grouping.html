폼 요소 그룹핑

1. 캡션
: <lable> 태그로 캡션과 폼 요소를 한 단위로 묶음
캡션과 폼 요소가 서로 연결 되어 있음을 브라우저가 알 수 있음
마우스 사용자가 더 사용하기에 편해짐

2. 캡션 사용 방법
1) <lable> 요소 안에 <input> 요소 넣기
2) <lable> 따로 <input> 따로 한 후 서로 연결

3. 입력 요소 그룹핑
: 하나의 폼 안에 여러 구역을 나누어 표시 할 때

  • <fieldset></fieldset> : 다양한 입력요소를 묶은 그룹의 경계에 선을 그려줌
  • <legend></legend> : 묶은 그룹에 제목을 붙여줌
