이미지 태그

<img> 태그 사용 : 단일 태그
웹 페이지에서 사용 할 수 있는 이미지 파일의 종류 : BMP, GIF, PNG, JPG(JPEG)

1. src 속성
1) 내 컴퓨터에 이미지 파일이 있을 때 : 절대, 상대 경로로 지정

2) 웹 상의 이미지 링크를 사용 할 때 : 크롬 - 이미지 찾기 - 오른쪽 마우스 클릭 - 이미지 주소 복사
단, 인터넷에 접속 할 수 있어야 화면에 표시가 가능, 저작권 주의

2. width, height 속성
: 이미지 크기 조정, 사용 안 할 때는 원본 이미지 크기 그대로 표시

3. alt 속성
: 어떤 이유로 이미지를 화면에 표시하지 못했을 경우 표시되는 대체 텍스트 지정

4. 이미지 관련 태그
1) <figure> : 이미지, 오디오, 비디오, 텍스트, 표 등 독립적인 콘텐츠를 담는 컨테이너 역할을 하는 태그

2) <figcaption> : figure 태그 영역 안에서 사용하며 설명 문구를 담는 태그
-> 하나의 <figure> 안에는 하나의 <figcaption> 사용
