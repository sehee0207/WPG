폼 요소(1)

1. 텍스트 입력
1) type="text" : 사용자가 입력 할 수 있는 한 줄 짜리 입력 필드
2) type="password" : 입력한 글자가 보이지 않음
3) <textarea> : 여러 줄 텍스트 입력 필드, cols 열 개수, rows 행 개수

2. 데이터 목록
<datalist> : 목록 리스트를 작성하는 태그

3. 버튼
1) <input type="button / reset / submit / image / file" value="버튼의 문자열">
2) <button type="buttom / reset / submit">버튼의 문자열</button>
-> 버튼을 만드는 방법은 두 가지, 결과적으로는 같은 버튼이 만들어짐
* 예시는 모두 첫 번째 방법을 사용 할 예정

4. 선택형 버튼
1) 라디오 버튼 <input type="radio">
: 여러 항목 중 하나만 선택, name 및 value 속성 반드시 지정
*name 이 같아야 동일한 그룹으로 취급 됨 *

2) 체크박스 <input type="checkbox">
: 여러 개의 항목을 동시에 선택
미리 체크해놓고 싶으으면 checked 속성 지정

3) 콤보박스 <select> </select>
: 드롭 다운 리스트에 목록을 출력해 선택하는 입력 방식
