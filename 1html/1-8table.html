테이블 태그

: 웹 문서에서 자료를 구조화하여 정리 할 때 사용
행(row), 열(column), 셀(cell)

  • <table> : 표 전체를 담는 컨테이너
  • <tr> : 행, 여러 td th 포함
  • <td> : 데이터 셀
  • <th> : 열 제목 셀
  • <caption> : 표 제목
  • <thead> : 헤딩 셀 그룹
  • <tfoot> : 바닥 셀 그룹
  • <tbody> : 데이터 셀 그룹
1. 테이블 만드는 방법
  1. <table>태그로 표 자리 만들기
  2. <tr>태그로 행 만들기
  3. <tr>태그 안에 <td></td>태그로 각 항목마다 셀 만들기
  4. 셀에 들어갈 내용은 <td></td>
2. 행 또는 열 합치기
행 합치기 : rowspan, 열 합치기 : colspan
<td>, <th> 태그의 속성

3. 표에 제목 붙이기
<table> 태그 바로 다음에 사용
표의 위쪽 중앙에 표시

4. 표 구조 정의하기
: 화면 낭독기를 통해 표의 구조를 쉽게 이해 가능
<thead> : 제목, <tbody> : 본문, <tfoot> : 요약
