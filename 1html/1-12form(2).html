폼 요소(2)

1. 검색 상자
input type="search" : 포털 사이트 검색, 페이지 내에서의 검색

2. 이메일 주소 입력
input type="email" : 유효한 이메일 주소인지 검사 가능
제출 버튼 눌렀을 때 시행, 잘못된 메일 주소시 경고 메시지

3. 숫자 입력 버튼
1) 스핀 버튼 input type="number" : 입력한 내용을 숫자로 인식
-> 속성 max, min, step, value (초기값)
2) 슬라이드 바 input type="range" : 슬라이드 막대로 숫자 지정
-> 속성 min, max, step, value

4. 날짜 입력

  • type="date" : 날짜 입력
  • type="datetime-local" : 날짜와 시간 입력
  • type="month" : 연도, 월 입력
  • type="week" : 연도, 주 입력
  • type="time" : 시간 입력
5. 색상 입력
input type="color" : 색상 선택

6. 추가된 <input>의 속성
  • autofocus : 자동으로 입력 포커스
  • placeholder : 입력 필드 안에 힌트를 희미하게 보여줌
  • readonly : 읽기 전용 필드 (수정불가)
  • required : 반드시 채워져 있어야 할 필수 피드 지정
7. 예제 ;

3. 생일은 날짜만 입력
