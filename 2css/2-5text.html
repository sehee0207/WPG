[CSS]
텍스트 스타일

1. color : 텍스트 색상
- 지정 방법
  1. 이름으로 표현 : { color : "red"; }
  2. 16진수 코드로 표현 : { color: "#FF0000";}
  3. rgb()로 표현_10진수 코드 또는 퍼센트로 표현 : { color : rgb(255, 0, 0);}
2. text-align : 텍스트 정렬
-> left, center, right, justify

3. text-decoration> : 텍스트 장식
-> none, overline, line-through, underline

4. text-transform : 텍스트 대소문자 변환
-> none, uppercase, lowercase, capitalize (차례대로 그대로, 대문자, 소문자, 대문자소문자 반대)

5. text-shadow : 텍스트 그림자
-> 강조효과, 입체적 효과
{ text-shadow: 가로거리(x방향), 세로거리(y방향), 번짐정도, 색상; }
  • 필수요소 -> 가로거리(x방향), 세로거리(y방향)
  • 여러 개의 그림자 효과 가능( , 로 구분)
6. 다중 컬럼 : 다단 설정
  • column-count : 컬럼 개수
  • column-gap : 컬럼 사이의 간격
  • column-rule: 구분선 스타일(구분선 두께, 스타일, 색상)
7. line-height : 줄 간격 지정
-> 보통 글자크기의 1.5 ~ 2 배면 적당

- 사용 예시
p {font-size: 15px; line-height:30px;}
p {font-size: 15px; line-height:2.0;}
p {font-size: 15px; line-height:200%;}
