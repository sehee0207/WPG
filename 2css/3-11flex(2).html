flex(2)

1. Flex item의 속성
1) flex-grow : 플렉스 항목의 증가 너비 비율 설정

2) flex-shrink: 플렉스 항목의 감소 너비 비율 설정
  • 플렉스 컨테이너의 크기가 항목보다 줄어들 때 flex-basis값을 가지고 있는 플렉스 항목의 너비를 줄이기
  • 속성 값 : 줄일 비율 (0: 줄이지 않음)
3) flex-basis : 플렉스 항목의 주축 방향의 기본 너비 지정

4) flex(위 3개 속성 축약형) : flex-grow, flex-shrink, flex-basis 한번에 지정
{ flex : flex-grow flex-shrink flex-basis; } 순서로

5) align-slef : 개별 플렉스 항목 별로 교차축 기준의 배치 방법 지정(개별적으로 지정)
-> 플렉스 항목 전체의 배치방법을 결정하는 컨테이너의 속성인 align-itmes 속성보다 우선순위가 높음
  • auto : 플렉스 컨테이너의 align-itmes 속성을 상속받음 (기본값)
  • stretch : 플렉스 항목을 확장하여 교차축 전체를 채움 (기본값)
  • flex-start : 플렉스 항목을 교차축의 시작점을 기준으로 배치
  • flex-end : 플렉스 항목을 교차축의 끝점을 기준으로 배치
  • center : 플렉스 항목을 교차축의 중앙을 기준으로 배치
6) order : 플렉스 항목의 배치 순서 설정하기
  • 기본값 : 0
  • 숫자가 클 수록 순서가 뒤로 밀림
