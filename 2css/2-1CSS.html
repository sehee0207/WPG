CSS

1. CSS
: Cascading Style Sheets

  • CSS는 1994년 10월 10일, 하콤 비움 리(Håkon Wium Lie)가 처음 제안
  • 마크업 언어가 실제 표시되는 방법을 기술하는 언어
  • 마크업 언어(ex: HTML)가 웹사이트의 몸체를 담당한다면 CSS는 옷과 액세서리처럼 꾸미는 역할을 담당
  • CSS로 작성된 코드를"스타일 시트(style sheet)"라고 부름
2. CSS3 스타일 시트 문법
선택자 { 속성: 값; 속성: 값; ••• } /* 주석문 */
  • 끝에 반드시 ';'
  • 주석 : /* 주석 내용을 적어주세요*/
  • 대소문자 구분 없음
