[CSS]
경계선 스타일

1. border-style : 경계선 스타일
-> 값 : none / dotted / dashed / solid / double / groove / ridge / inset / outset 등

2. border-width : 경계선의 폭
-> 값: ~px , 키워드(thin / medium / thick)

3. border-color : 경계선의 색상

4. border : 속성 축약
-> { border : border-width border-style border-color }

5. border-radius : 둥근 경계선
