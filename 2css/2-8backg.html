[CSS]
백그라운드 스타일

1. background-color : 배경색

2. background-iamge : 배경 이미지로 지정
-> background-image : url("이미지경로");

3. background-repeat : 배경이미지 반복 여부

repeat repect-x repect-y no-repeat
가로,세로로 반복 가로로 반복 세로로 반복 X

4. background-attachment : 배경이미지 부착 방법 지정
  • scroll : 배경이 요소와 같이 스크롤 됨 (디폴트값)
  • fixed : 배경이 뷰 포트에 대하여 고정됨
5. background-size : 배경 이미지 크기 설정
-> px단위 또는 퍼센트 단위로 지정 가능 (기준 : 부모 요소의 가로, 세로)

6. background-position : 배경 이미지 위치 설정
  • 값 : 수평위치 수직위치
  • 수평위치 : left / center / right / 백분율 / 길이값(px)
  • 수직위치 : top / center / bottom / 백분율 / 길이값(px)
7. background
  • 배경 이미지에 대한 속성 한 줄로 설정하기
  • background : background-color background-iamge background-repeat background-position
