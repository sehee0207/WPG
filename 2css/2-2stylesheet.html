[CSS]
위치

1. CSS 스타일 시트 위치 3가지
- 내부 스타일 시트(internal style sheet)
- 인라인 스타일 시트(inline style sheet)
- 외부 스타일 시트(external style sheet)

  1. 내부 스타일 시트
    • 내부 스타일 시트는 HTML 안에 CSS 정의
    • <head> 태그 내에 <style>태그에 스타일 시트 지정
    • 여러 번 작성 가능
    • 웹 페이지 전체에 적용
    • 한 문서에만 해당되는 스타일을 지정할 때 사용

  2. 인라인 스타일 시트
    • HTML요소 안에 style 속성 사용
    • 각각의 HTML 요소마다 스타일을 지정하는 것 -> 해당 태그에만 스타일 적용
    • t스타일시트의 장점을 잃게 됨.... 꼭 필요한 경우에만 사용

  3. 외부 스타일 시트
    4. -> HTML 파일
    -> CSS 파일

    • 스타일 시트를 외부에 파일(.css)로 저장 -> 웹 페이지에서 스타일 파일을 불러 사용
    • 요소 안에 태그 사용(단일태그)
      •  <link href="mystyle.css"       type="text/css"           rel="stylesheet"> 
                                외부 스타일 파일의 위치   연결문서의 MME유형          현재문서와 외부파일의 관계
                                                        (css의 경우: text/css)      (stylesheet:스타일시트로 연결)
                                                        (js의경우 text/javascript)
    • 많은 페이지에 동일한 스타일을 적용 할 때 좋은 방법
2. CSS 스타일 시트 규칙
1) 스타일 상속
  • CSS3 스타일은 부모 태그로부터 상속
  • 부모 태그(부모 요소) : 자신을 둘러싸는 태그
  • <p>태그는 <em>의 부모 태그
  • <em> 태그의 출력 : 글자 크기는 25px, 글자 색은 부모 <p> 태그를 상속받아 green
2) 다중 스타일 시트
  • 하나의 요소에 대하여 외부,내부,인라인 스타일에서 서로 다르게 지정하고 있다면 어떤 스타일이 사용될까?
  • ex. <br>, <meta>
3) 스타일 합치기(cascading)
  • 태그에 적용 가능한 스타일
    1. 브라우저의 디폴트 스타일
    2. 외부 스타일 시트
    3. 내부 스타일 시트
    4. 인라인 스타일 시트
  • 스타일 합치기(cascading)
    1. 태그에 적용되는 모든 스타일이 합쳐지고, 동일한 스타일은, 순위가 높은 스타일이 우선 적용되는 규칙
