onblur, onfocus, radio ...

1. onblur와 onfocus - 포커스가 변경될 때

  • 포커스 (focus)
    • 포커스는 현재 키 입력에 대한 독접권
    • 브라우저는 포커스를 가지고 있는 HTML 태그 요소에 키 공급
  • onblur
    • 포커스를 잃을 때 발생하는 이벤트 리스너
      • EX) 다른 HTML 요소를 클릭하면 현재 HTML 요소는 포커스를 잃는다.
  • onfocus
    • 포커스를 얻을 때 발생하는 이벤트 리스너
      • EX) 현재 HTML 요소를 클릭하면, 현재 HTML 요소가 포커스를 얻는다.
  • focus() 메소드 : DOM 객체의 메소드. 포커스 부여
2. 라디오 버튼
-- 라디오 버튼 객체
  • <input type = "radio" >로 만들어진 라디오 버튼 DOM 객체
  • name 값이 동일한 객체들이 하나의 그룹을 이룸
    •                 <form>
                    <input type="radio" name="city" value="seoul"> 서울
                    <input type="radio" name="city" value="busan"> 부산
                    <input type="radio" name="city" value="incheon"> 인천
                </form>
  • 라디오 버튼 객체들 알아내기 -> name으로
    •                 var korea_city = document.getElementsByName("city"); // korea_city[0], korea_city[1], korea_city[2]
  • 선택된 라디오 버튼 알아내기 : radio 객체의 checked 프로퍼티 조사(checked 프로퍼티가 true 이면 선택된 버튼)
    •                 var korea_city = document.getElementsByName("city"); //korea_city[0], korea_city[1], korea_city[2]
    -> 이 경우 korea_city[1].checked = true 이면 부산이 선택된 것을 의미함
3. 체크박스
  • <input type="checkbox">
  • radio 객체와 달리 name 속성이 모두 다름
  • 체크박스 객체의 checked 프로퍼티 사용하여 체크여부 확인
4. select 객체와 onchange
  • select 객체는 <select> 태그로 만들어진 콤보 박스
    • -> option 객체는 <option>태그로 표현되는 옵션 아이템 
                        <select id="fruits">
                        <option value="str">딸기</option>
                        <option value="blue">블루베리</option>
                        <option value="apple">사과</option>
                    </select>
  • 선택된 옵션 알아내기 : select객체의 selectedIndex 프로퍼티 (선택된 옵션에 대한 인덱스를 담고 있음)
    •                     var sel = document.getElementById("fruits");
                    var index = sel.selectedIndex;
  • 옵션 선택하기 (2가지 방법)

    1. select객체의 selectedIndex 프로퍼티 사용(선택된 옵션 인덱스)
      2.  sel.selectedIndex=2; // 3번째 옵션 "사과" 선택
    2. option객체의 selected 프로퍼티 사용(true:선택됨/false:선택안됨)
      3. sel.options[2].selected = true; // 3번째 옵션 "사과 선택"
  • onchange 리스너
    • 선택된 옵션이 변경되면 select 객체의 onchange 리스너 호출
      • <select id="fruits" onchange="changeImage()">...</select>
