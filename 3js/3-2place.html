[JavaScript]
위치

1. JavaScript 위치

  1. 인라인 스타일

    2. :HTML 태그의 이벤트 리스너 속성에 작성

  2. 내부 스타일

    3. :태그에 작성

  3. 외부 스타일

    4. :자바스크립트 파일(.js)에 작성

1-1. 인라인 스타일

: HTML 태그의 이벤트 리스너에 자바스크립트 코드 작성

EX) 
    <button type="button" onclick="alert('반갑습니다')">버튼을 누르세요!</button>
1-2. 내부 스타일
  • 내부 또는 body 내부에 작성 가능 ( 코드 실행 순서가 달라짐 )
  • 웹 페이지 내에 여러 번 작성 가능
EX) 
    <script> alert('안녕하세용'); </script>
1-3. 외부 스타일

:자바스크립트 파일(.js)에 작성

  • 자바 스크립트 외부파일 생성
    • 확장자 .js 파일에 저장
    • <script> 태그없이 자바스크립트 코드만 저장
  • 여러 웹 페이지(html파일)에서 불러와서 사용
    • <script> 태그의 src 속성으로 파일을 불러 사용
EX) 
    <script src="myjs.js"></script>
2. 자바스크립트로 HTML 콘텐츠 출력하기
  • 자바스크립트로 HTML 콘텐츠를 웹 페이지에 직접 추가 가능
    • 코드가 실행되는 위치에 출력됨
    • document.write()
EX) 
    document.write("<h3>Welcome!</h3>");
