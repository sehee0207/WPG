JS

1. JavaScript 언어
: 단순히 링크 위주의 하이퍼텍스트로만 이루어진 웹 페이지의 동작을 향상시키기 위해 만들어진 프로그래밍 언어

  • 사용자 동작에 반응하기 위한 언어
  • 웹의 표준 프로그래밍 언어
  • HTML의 공식적인 스크립트 언어
2. JavaScript란?
  • 1995년 넷스케이프의 브렌던 아이크(Brendan Eich)가 개발
  • 이름 변천사

    :모카 -> 라이브스크립트 -> 자바 스크립트

  • 홍보효과를 위해서 '자바'라는 이름을 차용했을 뿐, 자바 언어와는 무관하다
  • ECMA이 ECMAScript라는 이름으로 표준 재정
  • ECMA-262 1st Edition부터 ECMA-252 6th Edition까지 버전 추가
3. Javascript 특징
  • HTML 문서에 내장될 수 있음
  • 인터프리터 언어
    • 인터프리터로 실행
    • 컴파일 필요 없음
  • 단순함
    • C언어 구조 차용
    • 배우기 쉬움
  • 객체 기반 언어, 함수 개념
4. JavaScript의 역할
  • 사용자의 입력 및 계산 처리
  • 웹 페이지 내용 및 스타일의 동적 제어

    • :HTML 태그의 속성, CSS 프로퍼티 값 동적 변경

  • 브라우저 제어

    • :브라우저 윈도우 크기와 모양 제어, 새 윈도우 열기/닫기, 다른 웹 사이트 접속 등

  • 웹 서버와의 통신
  • 웹 애플리케이션 작성

    • :캔버스 그래픽, 로컬/세션 스토리지 저장, 위치정보서비스 등 다양한 API 제공

5. 코딩 시 주의사항
  • 대, 소문자 구분하여 작성하기
  • 코드 한 줄 작성 후에는 세미콜론 쓰기 ;
  • 가독성을 위해 한 줄에 한 문장만 작성하기
  • 문자형 데이터 작성시 큰 따옴표(" ")와 작은따옴표(' ')의 겹침 오류 주의
  • 코드 작성시 중괄호{}와 소괄호()의 짝 맞추기
  • 주석은 // 또는 /* */로 표현
