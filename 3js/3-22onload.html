[JavaScript]
onload

1. onload

  • 웹 페이지의 로딩이 완료되면 window 객체에 load 이벤트 발생
  • onload 리스너 작성 방법
    1. window.onload = function(){ alert('onload'); };
    2. <body onload = "alert('onload');" >
    -> 이 둘은 같은 표현 (<body>에 onload 리스너를 만드는 것은 사실상 window.onload를 이용하는 것과 동일함)
2. 이미지의 로딩 완료와 onload
  • 이미지가 출력되는 과정
    • ->이미지가 파일로부터 로딩하는 과정 + 이미지 출력 되는 과정
  • 이미지가 로딩되기 전에는 이미지의 크기, 비트맵 등을 알 수 없음
3. Image 객체
  • Image 객체
    • <img> 태그에 의해 생성되는 DOM 객체
    • 자바스크립트로 이미지를 다룰 수 있게 함
    • Image 객체의 프로퍼티
      • src : 이미지 주소
      • width : 이미지 폭
      • height : 이미지 높이
      • complete : 이미지의 로딩 완료 여부 (true/false)
  • img 태그의 이미지 바꾸기

    • HTML
    <img id="myImg" src="apple.png">
    ⬇

    JavaScript 
    var myImg = document.getElementById("myImg"); 
    myImg.src = "banana.png";
4. 이미지 로딩시 주의할 점
  • 코드 수정하기 예시
    • ->이미지 로딩이 완료된 후에 width 값 사용하기 -> onload 리스너 사용 
            var myImg = document.getElementById("myImg");

            myImg.onload = function(){
                var width = myImg.width;
                var height = myImg.height;
                alert("이미지 크기 = " + width + " x " + height);
            }
                
            myImg.src = "banana.png"; // 이미지 로딩 완료 시 실행
