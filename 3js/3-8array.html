[JavaScript]
배열

1. 배열(Array)

  • 배열
    • 여러 개의 원소들을 연속적으로 저장
    • 전체를 하나의 단위로 다루는 데이터 구조
  • 배열 생성 예시
    • EX) 
     var cities = ["Seoul", "New York", "Paris"];
    • 0부터 시작하는 인덱스를 이용하여 배열의 각 원소 접근
2. 배열 만드는 방법
  • 배열 만드는 2가지 방법
    • []로 배열 만들기
    • Array 객체로 배열 만들기
2-1. 배열 만드는 방법 - []로 배열 만들기
  • []로 배열 만들기
    • []안에는 원소들의 초기 값 나열
      • +) 배열의 크기를 미리 지정하지 않음
      연속적인 공간 x -> 산재해 있어 마음대로 늘이거나 줄일 수 있음- 쓰기는 편하나 효율의 차이 존재
    • 배열 크기
      • +) 배열 크기는 고정되지 않고, 원소 추가 시 늘어남
2-2. Array 객체로 배열 만들기
  • 초기 값을 가진 배열 생성
    • EX) 
     var week = new Array("월", "화", "수", "목", "금", "토", "일");
  • 초기화 되지 않은 배열 생성
    • 일정 크기의 배열 생성 후 나중에 원소 값 저장
      • EX) var seek = new Array(7); // 7개의 원소를 가진 배열 생성
      --> 이후 각 인덱스를 이용해 배열에 값 저장
    • 원소 개수를 예상할 수 없는 경우
      • EX) var week = new Array()l // 빈 배열을 생성하고
      --> 그 이후에 인덱스를 이용해 값을 넣어주면 자동으로 배열의 크기를 잡아줌
3. 배열의 원소 개수, length 프로퍼티
  • 자바스크립트의 모든 배열은 length 프로퍼티가 있다.
  • length 프로퍼티 : 배열 내에 가장 큰 인덱스에 1을 더한 값 (원소의 개수와 일치하지는 않음)
  • length 프로퍼티는 사용자가 임의로 값 변경 가능
    • length 프로퍼티는 Array 객체에 의해 자동 관리
    • 사용자가 임의로 값 변경 가능
      • 배열의 크기를 줄이거나 늘일 수 있음
        • EX) 
         var plots = [-20, -5, 0, 15, 20];
    console.log(plots.length); // 배열의 길이 출력
    plots.length = 10; // 배열의 크기를 10으로 늘림
    console.log(plots) // [-20, -5, 0, 15, 20, undefined*5] 출력
    plots.length = 2 // 배열의 크기를 2로 줄임
    console.log(plots); // [-20. -5] 출력
4. 자바스크립트 배열의 특징
  • 배열에 여러 타입(정수, 실수, 문자열, 논리값, 객체)의 데이터 섞어 저장 가능
  • 배열은 Array 객체
    • []로 생성해도 Array 객체로 다루어짐
5. Array 객체의 메소드 (사용법: 배열이름.메소드();)
  • pop()
    • 배열에 저장된 데이터 중 마지막 인덱스 저장 값을 삭제, 현재 배열이 변함
  • push(새 데이터)
    • 배열 객체의 마지막 인덱스에 '새 데이터' 삽입, 현재 배열이 변함
  • shift()
    • 배열 객체에 저장된 데이터 중 첫번째 인덱스에 저장된 데이터 삭제, 현재 배열이 변함
  • unshift(새 데이터)
    • 배열 객체의 첫번째 인덱스에 '새 데이터' 삽입, 현재 배열이 변함
  • reverse()
    • 배열 객체의 데이터 순서를 역순으로 바꿈, 현재 배열이 변함
  • sort()
    • 배열 객체의 데이터를 오름차순으로 정렬, 현재 배열이 변함
  • concat(다른 배열)
    • 현재 배열에 다른 배열의 원소들을 덧붙여 만든 새 배열을 리턴
  • join(연결 문자)
    • 배열 객체의 데이터를 ‘연결 문자’로 이어 붙여 1개의 문자형 데이터를 리턴.
      배열을 서버로 보낼 때 유용. 연결문자’ 생략 시 “,”이용. 현재 배열은 변하지 않음.
  • slice(index1, index2)
    • 배열 객체의 데이터 중 원하는 인덱스 구간만큼 잘라서 새 배열을 리턴, 현재 배열은 변하지 않음
      • +) index1 생략시 0부터, index2 생략시 끝까지
  • splice(시작 인덱스, 삭제할 데이터 개수, 추가할 데이터)
    • 배열 객체의 '시작인덱스'부터 '삭제할 데이터 개수'만큼 삭제하고 구 자리에 '추가할 데이터'를 삽입 '삭제할 데이터 개수' 생략 시 끝까지 삭제
      '추가할 데이터' 생략시 삭제만 수행, 현재 배열이 변함
