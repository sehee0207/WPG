1. HTML 태그의 요소

  • HTML 태그(Element라고도 불림)
    • 엘리먼트 이름(태그 이름)
    • 속성(프로퍼티)
    • CSS3 스타일
    • 이벤트 리스너
    • 콘텐츠(innerHTML)
2. DOM 객체의 구성 요소
: DOM 객체는 5개의 요소로 구성(HTML 태그의 구성요소를 그대로 반영)
  • 프로퍼티(property)
    • DOM객체의 멤버 변수로서, HTML 태그의 속성 반영
    • DOM 객체의 프로퍼티 값으로부터 HTML 태그의 속성을 알아내거나, 변화를 줄 수 있음.
  • 메소드(method)
    • DOM 객체의 멤버 함수로서, HTML 태그 제어에 활용
  • 컬렉션(collection)
    • 정보를 집합적으로 표현하는 일종의 배열
    • 예) children 컬렉션 : 모든 자식 DOM 객체들의 주소를 가짐
  • 이벤트 리스너(event listener)
    • HTML 태그에 작성된 이벤트 리스너 반영
  • CSS3 스타일(style)
    • HTML 태그에 설정된 CSS3 스타일 시트 정보를 반영
    • DOM 객체의 style 프로퍼티를 통해 HTML 태그의 모양 제어 가능
3. DOM 객체의 주요 프로퍼티
  • 기본
    • id
      • 태그의 id 속성 값
    • lang
      • 태그의 lang 속성 값
    • style
      • style 객체에 대한 레퍼런스
    • title
      • 태그의 title 속성 값
    • tagName
      • 태그 이름 문자열(항상 대문자로 표시)
    • innerHTML
      • 시작태그와 종료태그 사이에 HTML
  • DOM 트리 관련
    • childElementCount
      • 자식 DOM 객체 개수
    • firstElementChild
      • 첫번째 자식 객체
    • lastElementChild
      • 마지막 자식 객체
    • nextElementSibling
      • 다음 형태(오른쪽) 객체
    • parentElement
      • 부모 객체
    • previousElementSibling
      • 이전 형태(완쪽) 객체
  • 크기와 위치
    • offsetHeight
      • 패딩, 테두리, 스크롤바를 포함한 높이
    • offsetWidth
      • 패딩, 테두리, 스크롤바를 포함한 폭
    • offsetLeft
      • 객체의 출력 위치. 수평 옵셋
    • offsetTop
      • 객체의 출력 위치. 수직 옵셋
  • 컬렉션
    • children
      • 자식 DOM객체들이 컬렉션 ( 자식들을 가리키는 레퍼런스 배열)
        • --> 직계 자식들의 컬렉션
4. DOM 객체의 주요 메소드
  • addEventListener()
    • 새로운 이벤트 리스너 등록
  • removeEventListener()
    • addEventListener()로 등록한 이벤트 리스너 제거
  • click()
    • 사용자가 마우스를 클릭한 것과 동일한 작업 수행
  • focus()
    • 키 입력을 받을 수 있도록 이 객체에 포커스 지정
  • setAttribute()
    • 속성 추가
  • insertBefore()
    • 지정된 자식 앞에 새 자식 추가
  • removeChild()
    • 자식 삭제
  • replaceChild()
    • 자식 대체
  • appendChild()
    • 마지막 자식 뒤에 새로운 자식 추가

      : font-size 스타일 프로퍼티 -> fontSize

7. this
    this 키워드
    • DOM 객체에서 객체 자신을 가리키는 용도로 사용
      • 예) div 태그 자신의 배경색을 오렌지 색으로 변경
        • -EX) 
        <div onclick = "this.style.backgroundColor='orange'">
      • 예) 버튼이 클릭되면 자신의 배경색을 오렌지색으로 변경
        • -EX) 
         <button onclick="this.style.backgroundColor='orange'">
8. 문서의 동적 구성
  1. DOM 객체 동적 생성 : document.createElement("태그 이름")
    • 새로운 DOM 객체 생성 (태그, 속성 부여)
      • EX)
          var newDIV = document.createElement("div");
    newDIV.innerHTML = "새로 생성된 DIV입니다";
    newDIV.setAttribute("id", "myDiv"); // 속성 추가
    newDIV.style.backgroundColor = "yellow";
      --> 위 코드와 같음 
       <div id = "myDiv" style = "background-color:yellow"> 
        새로 생성된 DIV 입니다. </div>
  2. DOM 트리에 추가 : 부모.appendChild(DOM객체);
    • 예) 새로 생성한 div 객체를 <p id = "p"> 태그의 마지막 자식을 추가
      • EX) 
       var p = document.getElementById("p"); // 아이디가 p인 DOM 객체 찾기
      p.appendChild(newDIV); // 객체 p의 마지막자식으로 newDIV 추가
  3. DOM 객체의 삭제 : 부모.removeChild(삭제하고자하는 자식 객체);
    4. EX)
            var myDiv = document.getElementById("myDiv"); //자식 DOM 객체 찾기
        var parent = myDic.parentElement; // 부모 DOM 객체 찾기

        parent.removeChild(myDiv); // 부모 객체에서 myDiv 객체 삭제 및 반환
