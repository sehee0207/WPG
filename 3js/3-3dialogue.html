[JavaScript]
다이얼로그

1. 프롬프트 다이얼로그

: prompt("메세지", "디폴트 입력값") 함수

  • 사용자로부터 문자열을 입력 받아 리턴
  • "디폴트 입력값"은 생략 가능함
2. 확인 다이얼로그

: confirm("메시지") 함수

  • "메세지"를 출력하고 '확인/취소(OK/CANCEL)'버튼을 가진 다이얼로그 출력
  • '확인'버튼을 누르면 true 값을, '취소'버튼이나 강제로 다이얼로그를 닫으면 false 값을 리턴
3. 경고 다이얼로그

: alert("메세지") 함수

  • '메세지'와 '확인' 버튼을 가진 다이얼로그 출력, 메세지 전달
